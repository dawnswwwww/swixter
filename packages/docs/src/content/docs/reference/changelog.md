# Changelog

Full version history is maintained in the [CHANGELOG.md](https://github.com/dawnswwwww/swixter/blob/main/CHANGELOG.md) on GitHub.

## Latest Version

The current stable version is displayed in the CLI:

```bash
swixter version
```

## Recent Releases

See the [GitHub Releases page](https://github.com/dawnswwwww/swixter/releases) for detailed release notes, downloads, and upgrade instructions.

## Versioning

Swixter follows [Semantic Versioning](https://semver.org/):

- **MAJOR** (`1.0.0`): Breaking changes
- **MINOR** (`0.1.0`): New features, backward-compatible
- **PATCH** (`0.0.1`): Bug fixes, backward-compatible

## Release Channels

Swixter is published to npm. Install the latest stable version:

```bash
npm install -g swixter
```

For pre-release versions, check the [GitHub Releases](https://github.com/dawnswwwww/swixter/releases) page for tags and assets.
